把 GLM-TTS 生成的音频放在这里（文件名为 `数字值.wav`），应用会优先播放本地音频：

- `audio/10.wav`（十颗糖）
- `audio/100.wav`（一百颗糖）
- `audio/1000.wav`（一千颗糖）
- `audio/10000.wav`（一万颗糖）
- `audio/100000.wav`（十万颗糖）
- ……
- `audio/1000000000000.wav`（一万亿颗糖）
- `audio/10000000000000.wav`（十万亿颗糖）
- `audio/100000000000000.wav`（一百万亿颗糖）
- `audio/1000000000000000.wav`（一千万亿颗糖）
- `audio/10000000000000000.wav`（一亿亿颗糖）

生成脚本：`tools/generate-audio.mjs`
